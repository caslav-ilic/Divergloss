Package dg

Module for handling Divergloss XML glossaries.

Author: Chusslove Illich (Часлав Илић) <caslav.ilic@gmx.net>

License: GPLv3

Submodules
  • dg.construct: Construct internal glossary out of a Divergloss XML document.
  • dg.dset: Handler for glossary parts which can differ by language and environment.
  • dg.filter: Glossary filters.
    • dg.filter.pass: Filter which just passes glossary unchanged.
    • dg.filter.sr_latin: Transform Serbian Cyrillic text in the glossary into Serbian Latin.
  • dg.query: Methods for selection data within the glossary.
  • dg.subcmd: Handle subcommands and their parameters.
  • dg.textfmt: Format text in glossary for different outputs.
  • dg.util: Various utilities.
  • dg.view: Glossary views.
    • dg.view.null: View which shows nothing of the glossary.
    • dg.view.po: Create a PO file out of the glossary.
    • dg.view.text_simple: A simple text view (just an example).
